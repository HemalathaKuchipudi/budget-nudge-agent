# Budget Nudge Agent 💰

Budget Nudge Agent is a web application that helps students track their spending and encourages smarter financial habits.

## Problem
Many students struggle with managing their daily expenses and often overspend.

## Solution
This application analyzes spending inputs and provides simple budgeting suggestions to help users improve their financial discipline.
